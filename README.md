# @zenuml/repl

A simple REPL (Read-Eval-Print Loop) for ZenUML.

## Simple Usage

```javascript
A.method() {
  B.method()
  if(condition) {
    C.method()
  }
}
```

## How does it work

In the Preview component, we create an `iframe` on the page with
src to be 'https://embed.zenuml.com/embed.html'.
